<TabItem value="list">

Return a list of effective tags for the given Google Cloud resource, as specified in `parent`.

```sql
SELECT
effectiveTags,
nextPageToken
FROM google.cloudresourcemanager.effective_tags
WHERE parent = '{{ parent }}'
AND pageSize = '{{ pageSize }}'
AND pageToken = '{{ pageToken }}';
```
