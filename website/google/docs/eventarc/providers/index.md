## `SELECT` examples

<Tabs
    defaultValue="get"
    values={[
        { label: 'get', value: 'get' },
        { label: 'list', value: 'list' }
    ]}
>
<TabItem value="get">

Get a single Provider.

```sql
SELECT
name,
displayName,
eventTypes
FROM google.eventarc.providers
WHERE projectsId = '{{ projectsId }}' -- required
AND locationsId = '{{ locationsId }}' -- required
AND providersId = '{{ providersId }}' -- required;
```
</TabItem>
<TabItem value="list">

List providers.

```sql
SELECT
nextPageToken,
providers,
unreachable
FROM google.eventarc.providers
WHERE projectsId = '{{ projectsId }}' -- required
AND locationsId = '{{ locationsId }}' -- required
AND pageSize = '{{ pageSize }}'
AND pageToken = '{{ pageToken }}'
AND orderBy = '{{ orderBy }}'
AND filter = '{{ filter }}';
```
</TabItem>
</Tabs>
