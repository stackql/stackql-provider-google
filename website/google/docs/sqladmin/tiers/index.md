Lists all available machine types (tiers) for Cloud SQL, for example, `db-custom-1-3840`. For more information, see https://cloud.google.com/sql/pricing.

```sql
SELECT
items,
kind
FROM google.sqladmin.tiers
WHERE project = '{{ project }}' -- required;
```
