## `SELECT` examples

<Tabs
    defaultValue="get"
    values={[
        { label: 'get', value: 'get' },
        { label: 'list', value: 'list' }
    ]}
>
<TabItem value="get">

Retrieves specified company.

```sql
SELECT
name,
careerSiteUri,
derivedInfo,
displayName,
eeoText,
externalId,
headquartersAddress,
hiringAgency,
imageUri,
keywordSearchableJobCustomAttributes,
size,
suspended,
websiteUri
FROM google.jobs.companies
WHERE projectsId = '{{ projectsId }}' -- required
AND tenantsId = '{{ tenantsId }}' -- required
AND companiesId = '{{ companiesId }}' -- required;
```
</TabItem>
<TabItem value="list">

Lists all companies associated with the project.

```sql
SELECT
name,
careerSiteUri,
derivedInfo,
displayName,
eeoText,
externalId,
headquartersAddress,
hiringAgency,
imageUri,
keywordSearchableJobCustomAttributes,
size,
suspended,
websiteUri
FROM google.jobs.companies
WHERE projectsId = '{{ projectsId }}' -- required
AND tenantsId = '{{ tenantsId }}' -- required
AND pageToken = '{{ pageToken }}'
AND pageSize = '{{ pageSize }}'
AND requireOpenJobs = '{{ requireOpenJobs }}';
```
</TabItem>
</Tabs>

```sql

SET 
data__name = '{{ name }}',
data__displayName = '{{ displayName }}',
data__externalId = '{{ externalId }}',
data__size = '{{ size }}',
data__headquartersAddress = '{{ headquartersAddress }}',
data__hiringAgency = {{ hiringAgency }},
data__eeoText = '{{ eeoText }}',
data__websiteUri = '{{ websiteUri }}',
data__careerSiteUri = '{{ careerSiteUri }}',
```
