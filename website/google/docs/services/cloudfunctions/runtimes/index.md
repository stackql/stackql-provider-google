Returns a list of runtimes that are supported for the requested project.

```sql
SELECT
runtimes
FROM google.cloudfunctions.runtimes
WHERE projectsId = '{{ projectsId }}' -- required
AND locationsId = '{{ locationsId }}' -- required
AND filter = '{{ filter }}';
```
